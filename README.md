# ServerControllerClient

A lightweight remote Minecraft server controller built with:

- Python/PySide6 desktop client
- Cross-platform GUI client

Designed for my friends to remotely controlling a Minecraft server running on a different computer.
**BE ADVISED THAT ONLY ONE CLIENT CAN BE CONNECTED AT A TIME**

---

# Features

- Secure TCP authentication
- Wake-on-LAN support
- Custom TCP command handling
- Python GUI client
- Cross-platform desktop client
- Config-driven backend
- Designed for client device
  
Check Issues for any known issues
